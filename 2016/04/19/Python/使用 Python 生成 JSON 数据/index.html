使用 Python 生成 JSON 数据

代码

这个脚本本来是用来模拟返回的数据，想着最近学了一点 Python，就想通过用 Python 来生成一堆 JSON 格式的数据

返回数据的需求：

  1. 包含一个状态码字段，用来指示返回数据的状态，这里默认设为 0000
  2. 包含一个状态描述字段，用来描述返回数据的状态，这里默认设为 服务执行成功
  3. 包含一个内容的字段，这个字段包含我需要的东西，固然也是一个数组了

于是，我们有了3个字段：errorcode, errormsg, ‘bodys’，分别对应上述的3个字段
其中，我们这个模拟数据所包含的内容是多个邮件的内容，而邮件里面包含了附件的信息，于是，使用类 ReturnInfo 来表示一次返回的数据的模型，ReturnBody 表示每一封邮件的信息，AttachListInfo 表示每一封邮件中的附件信息
