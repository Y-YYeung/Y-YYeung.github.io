  • 属性名前加上双下划线 __ ，可变为私有变量
  • 强制访问私有变量：<a instance>._<class name>__<property name>，如：bart._Student__name
  • 双下划线开头，并以双下划线结尾，为特殊变量，可直接访问

特殊变量

__str__

__iter__

如果一个类想被用于for … in循环，类似list或tuple那样，就必须实现一个iter()方法，该方法返回一个迭代对象，然后，Python的for循环就会不断调用该迭代对象的next()方法拿到循环的下一个值，直到遇到StopIteration错误时退出循环。

__getitem__

要表现得像list那样按照下标取出元素，需要实现getitem()方法

__call__

实例本身上直接调用方法
