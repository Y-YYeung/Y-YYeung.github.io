Linux 中 profile, bashrc, bash.profile

By Wen Yongyang

发表于 2015-11-23

文章目录

  • etc/profile

    • 为系统每个用户设置环境信息
    • 用户第一次登录时被执行
    • 并从 /etc/profile.d 目录的配置文件中搜集shell的设置
    • 需重启生效

  • /etc/bashrc

    • 为每一个运行 bash shell 用户执行此文件
    • bash shell 被打开时，该文件被读取

  • ~/.bash_profile

    • 每个用户使用该文件设置专用于自己的 shell 信息
    • 用户登录时，改文件仅执行一次
    • 需重启生效

  • ~/.bashrc

    • 含专用于你的 bash shell 的 bash 信息
    • 登录时以及每次打开新的 shell 时,该文件被读取
    • 每个用户都有一个 .bashrc1 文件，在用户目录下

  • ~/.bash_logout

    • 每次退出系统(退出bash shell)时,执行该文件
